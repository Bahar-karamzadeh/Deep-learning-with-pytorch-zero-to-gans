# Deep Learning with PyTorch: Zero to GANs

This repository contains assignments and project work completed as part of the
**Deep Learning with PyTorch: Zero to GANs** course on Jovian. The course provides
a practical, beginner-friendly introduction to deep learning using PyTorch.

## Course Overview
The course focuses on learning deep learning concepts through hands-on coding
using Jupyter notebooks. Topics include tensors, gradients, neural networks,
and modern deep learning architectures.

## Repository Contents

### Assignment 1 – PyTorch Tensor Operations
- Exploring PyTorch tensors and core tensor functions
- Working and failing examples for clarity
- Publishing and sharing Jupyter notebooks

### Assignment 2 – Train Your First Model
- Loading and exploring real-world datasets
- Building and training models using PyTorch
- Evaluating models and making predictions

### Assignment 3 – Feed Forward Neural Networks
- Training deep neural networks
- Using GPUs for faster training
- Hyperparameter tuning and optimization

## Final Course Project
- Training a deep learning model from scratch
- Working with a large real-world dataset
- Documenting and publishing results

## Prerequisites
- Basic programming (functions and loops)
- Basic linear algebra
- Basic calculus
- No prior deep learning experience required
